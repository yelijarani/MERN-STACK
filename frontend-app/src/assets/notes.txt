  COMPONENTS IN  React:
  ------------------
    ->components are the smallest UI blocks called as component

    TYPES:
    ------
    1.Class Based components
         ->componentare declared by using class keyword is called as class based components
         ->In CBC constructions,superkeyword,this keyword is present also render() , and life cycle methods are present
         ->in CBC states are present so that's why is called as a statefull components

         SYNTAX:
         ---------
                   import {Component} from react;               
                   class ClassName extends Components{
                    render(){
                        return(
                            html code
                        )
                    }
                   }
                   export default ClassName;

    2.Function based component(FBC)
        ->We can declared FBC using arrow function or normal function or function expression 
        ->IN fbc constructors,superkeyword,thiskeyword not present
        ->Render() and life cycles methods also not present
        ->In FBC state are not present so thats why is called as sateless components

    SYNTAX1:
    -------
                 const ComponentName=()=>{
                    return{
                        html codehere..

                    }
                 }
                 export default ComponentName;

    SYNTAX2:
    --------
                function ComponentName{
                    {
                        return(
                            html code here... 

                        )
                   }
                    
                }
                export default ComponentName;

     SYNTAX3:
     --------
                function ComponentName=function{
                    {
                        return(
                            html code here... 

                        )
                   }
                    
                }
                export default ComponentName;
 STATES IN REACT:
 ----------------
        ->States are used to hold the data
        ->States are mutable in nature
        ->States are present only in CBC

HOOKS:
-----
     -<Hooks are used to make function based components state full
     ->hooks are used only FBC
     ->hooks are introduced in 16.8 react version 
     EX:
       1.useState()
       2.useEffect()
       3.useNavigate()
       4.useMemo()
       5.useContext()
       6.useReduce()
       7.useRef()---etc

STATES IN FUNCTION BASED COMPONENTS:
-----------------------------------
       ->we can declared state in FBC using useState() hook 


PROPS(properties):
-----------------
   ->Props are used to transfer data from one Component to another component
   ->Props are immutable in nature
   ->Props are present both in CBC and FBC
