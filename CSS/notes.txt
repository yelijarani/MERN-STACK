lCSS:
-----
 ->CSS stands for cascading style sheet used to apply stylings,layouts,animations to the web page
 ->CSS are introduced in the year 1994 by the developer "hakon wium lie"
 ->the present version of CSS is CSS3


 Methods to apply stylings to wb web page:
 ----------------------------------------
 1.inline CSS/inline stylings:
    ->applying styling by taking 'style attribute' in HTML element
    ->this styling is applicable for only thst elements
    ->foe inline CSS not required CSS selectors
    ->inline styling is more priority that internal and external CSS
 2.internal:
    ->applying styles by taking <style></style> tag in head partis called as a internal stylings
    ->to apply internal CSS wde need CSS selectors 
    ->
 3.external:
    ->apply styling by taking new.CSS file and link to HTML file by using <link></link> tag
    -> apply external css we need CSS selectors


 CSS selectors
    ----------------
    ->css selectors are used to target hyml elements to applt the stylings
    ->css selectors are present only in internal and external selectors CSS 

TYPES OS CSS selectors
----------------------
1.simple selectors:
  ->ID selectors(#)
  ->class selectors(.)
  ->tag selectors 
  ->uping selectors(,)
  ->universal selectors(*)

2.combinator:
  ->desendent selectors(:)
  ->child selectors(space)
  ->adjacent sibling selectors(+)
  ->general siblingselectors (~)

3.pseudo class:
   i.structure pseudo class selectors:
    1. :first-child
    2. :last_child
    3. :first_of last_child4.
    4. :last_of _type
    5. :nth_child()
ii.dynamic pseudo class selectors:
    1.:active     ----
    2. :visited   ----  ->anchor tags
    3. :focus     ----  ->input tag
    4. :hover



  4.pseudo elements selectors
    ->::after
   ->::before
   ->::first-letter
    ->::first-line
    ->::marker
    ->::selection
    ->::placeholder

  5.attribute selectors:
    ->Attribute Selectors are used to target HTML Element based on attributes 
        Ex: <h1 algin="center" id="heading">  HELLO  </h1>
       <h1  id="heading">  HELLO  </h1>
          syntax:  selector[attributeName="value"]{
             .... styling.....
               }
            
            types
                1. [attributeName="value"] 
                2. [attributeName^="value"]
                3. [attributeName$="value"]
                4. [attributeName|="value"]
                5. [attributeName*="value"]
